Course Overview & Hardware Requirements

This book is structured as a comprehensive 13-week course, designed to progressively build your understanding and skills in Physical AI and Humanoid Robotics. To facilitate hands-on learning, various hardware tiers and cloud alternatives are discussed, allowing you to choose the setup that best fits your budget and aspirations.

13-Week Course Breakdown​

The course is divided into modules, each focusing on key aspects of Physical AI. Each week will typically involve reading chapters, working through code examples, and engaging with simulation exercises.

Part 0: Introduction (Weeks 1-2)

  • Chapter 0.1: What is Physical AI?
  • Chapter 0.2: Why Humanoid Robots?
  • Chapter 0.3: Course Overview & Hardware Requirements
  • Chapter 0.4: Environment Setup

Part 1: The Robotic Nervous System - ROS 2 (Weeks 3-5)

  • Chapter 1.1: ROS 2 Architecture & Core Concepts
  • Chapter 1.2: Nodes, Topics, and Services
  • Chapter 1.3: Building ROS 2 Packages with Python (rclpy)
  • Chapter 1.4: Understanding URDF for Humanoids
  • Chapter 1.5: Launch Files and Parameter Management
  • Chapter 1.6: Bridging Python Agents to ROS Controllers

Part 2: The Digital Twin - Gazebo & Unity (Weeks 6-7)

  • Chapter 2.1: Gazebo Simulation Environment Setup
  • Chapter 2.2: URDF and SDF Robot Description Formats
  • Chapter 2.3: Simulating Physics, Gravity, and Collisions
  • Chapter 2.4: Sensor Simulation (LiDAR, Depth Cameras, IMUs)
  • Chapter 2.5: High-Fidelity Rendering in Unity
  • Chapter 2.6: Human-Robot Interaction in Simulation

Part 3: The AI-Robot Brain - NVIDIA Isaac (Weeks 8-10)

  • Chapter 3.1: NVIDIA Isaac SDK and Isaac Sim Overview
  • Chapter 3.2: Photorealistic Simulation & Synthetic Data Generation

  • Chapter 3.4: Nav2: Path Planning for Bipedal Movement

Part 4: Vision-Language-Action - VLA (Weeks 11-13)

  • Chapter 4.1: Humanoid Robot Kinematics and Dynamics
  • Chapter 4.2: Bipedal Locomotion and Balance Control
  • Chapter 4.3: Manipulation and Grasping with Humanoid Hands
  • Chapter 4.4: Voice-to-Action with OpenAI Whisper
  • Chapter 4.5: Cognitive Planning with LLMs
  • Chapter 4.6: Multi-modal Interaction (Speech, Gesture, Vision)
  • Chapter 4.7: Integrating GPT Models for Conversational Robotics

Part 5: Capstone Project (Final Integration)

  • Chapter 5.1: Project Overview
  • Chapter 5.2: System Architecture Design
